# Logo IFRN — Configuração

Place the IFRN Campus Pau dos Ferros logo here as `if.jpeg`.

This file is referenced by the LaTeX templates and render_assessment.py.
The render script copies it to the build directory before running xelatex.

Typical source: the `if.jpeg` you already use in your LaTeX documents.

```
cp /path/to/if.jpeg skills/assessment-create/assets/if.jpeg
```
